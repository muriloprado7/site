Projeto



Este site tem como objetivo evidenciar que o sucesso empresarial sempre é possível com esforço e inovação. Isso é mostrado com startups de diversos ramos que "deram certo" e hoje são conhecidas por valiosos produtos e serviços, quantificados por um grande valuation(estimativa do valor de uma empresa).

Fica claro que, dentro desse objetivo, está inerente a vontade de inspirar os jovens brasileiros para sobressairem-se, e poderem, muitas vezes,modificar suas condições adversas de vida, proporcionando situações melhores, tanto para eles quanto para as famílias.

Logo, percebe-se que tal projeto abrirá portas para o povo brasileiro sintonizar e ver que todos podem ser os próximos a motivarem outros. Isso irá gerar uma maior quantidade de empregos decentes e consequetemente, um crescimento econômico inovador, que fazem parte tanto do Objetivo de Desenvolvimento Sustentável número 8 quanto do objetivo número 9.
